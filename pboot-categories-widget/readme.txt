Categories widget for bootstrap based themes! Display lists of categories in website sidebar.

== Description ==

Widget to display categories. Made for bootstrap 4. Based on WP Categories Widget by MR Web Solution http://www.mrwebsolution.in/ (http://raghunathgurjar.wordpress.com)


= `Features` =
 * Select categories you want to display
 * Optimized for sidebar view
 * Uses div classes: list-group, list-group-item, and badges classes for post counts

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The categories post will be available on widegt page (Apperance >> Widget)
    WordPress administration page.
